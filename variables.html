Общая цель страницы

Экран Variables используется администратором для управления параметрами pricing engine. Основной принцип экрана — прозрачность и контролируемость. Поэтому данные представлены таблицами, без лишних визуальных элементов.

Fuel Logic

Fuel Source

https://gasprices.aaa.com/

Таблица Fuel заполняется через backend integration / scheduled job. Данные должны сохраняться snapshot-ом.

Fuel Total Price

Total Price = Current Avg. × (1 + Cocharge % / 100)

Правила масштабирования

  • Все variables должны иметь группу, код, значение, unit, type и status.
  • Fuel prices являются внешними данными и не должны редактироваться вручную, кроме Cocharge %.
  • Изменение variables не должно менять уже отправленные estimates.
  • Каждый estimate должен сохранять variables_snapshot.
  • Все изменения должны попадать в audit log.
  • Новые группы variables добавляются отдельными таблицами, если количество параметров растёт.
